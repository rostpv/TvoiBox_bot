# Проверка production владельцем

Короткий чек-лист для владельца после деплоя.

Дата актуализации: 2026-06-03.

## Быстрая техническая проверка

- [x] `https://api.tvoybox.ru/health` открывается и возвращает `status: ok`.
- [x] `https://app.tvoybox.ru/` открывается.
- [x] `https://app.tvoybox.ru/mini-api/health` открывается и возвращает `status: ok`.

## Проверка Telegram

- [x] Открыть `@TvoyBox_bot`.
- [x] Нажать `/start`.
- [x] Проверить, что бот отвечает.
- [x] Проверить, что появилась кнопка Mini App.
- [x] Открыть Mini App внутри Telegram.

## Проверка клиента

- [x] Клиент видит главное меню.
- [x] Клиент может открыть запись на тренировку.
- [x] Клиент может отправить заявку на слот.
- [x] Клиент может открыть `Мои записи`.

## Проверка тренера

- [x] Тренер нажал `/start`.
- [x] Тренер видит тренерский экран.
- [x] Тренер видит заявку клиента.
- [x] Тренер может открыть или закрыть слоты.
- [x] Тренер может подтвердить заявку.
- [x] Тренер может отменить тренировку.
- [x] Тренер может перенести тренировку.
- [x] Тренер может настроить длительность тренировки.
- [x] Тренер может настроить рабочее время с минутами, например `08:15`.

## Проверка уведомлений и календаря

- [x] Клиент получает Telegram-уведомление после подтверждения.
- [x] Клиент получает или открывает `.ics` файл.
- [x] В Google Calendar появляется событие.
- [x] Отмена тренировки корректно обновляет запись и Google Calendar.
- [x] Перенос тренировки корректно обновляет запись и Google Calendar.
- [x] Повторная проверка 2026-06-03 после fix: перенос прошёл, старое событие исчезло или обновилось, нового дубля в Google Calendar нет.

## Проверка слотов с новой длительностью

- [x] В настройках тренера доступна длительность: 30, 45, 60, 75, 90, 105, 120 минут.
- [x] В режиме работы можно выбрать минуты в начале и конце дня.
- [x] При длительности 1 час 30 минут и рабочем времени `08:15-13:15` слоты строятся как полные интервалы внутри рабочего дня: `08:15-09:45`, `09:45-11:15`, `11:15-12:45`.
- [x] Google Calendar и `.ics` используют фактическую длительность тренировки.

## Что передать разработчику при проблеме

- время ошибки по Москве;
- Telegram username и Telegram ID пользователя;
- какой экран открыт;
- что нажимали;
- скриншот ошибки;
- открывалось внутри Telegram или в обычном браузере.

Не пересылать токены, пароли, приватные ключи и JSON service account в чат.
