# Проверка production владельцем

Короткий чек-лист для владельца после деплоя.

## Быстрая техническая проверка

- [ ] `https://api.tvoybox.ru/health` открывается и возвращает `status: ok`.
- [ ] `https://app.tvoybox.ru/` открывается.
- [ ] `https://app.tvoybox.ru/mini-api/health` открывается и возвращает `status: ok`.

## Проверка Telegram

- [ ] Открыть `@TvoyBox_bot`.
- [ ] Нажать `/start`.
- [ ] Проверить, что бот отвечает.
- [ ] Проверить, что появилась кнопка Mini App.
- [ ] Открыть Mini App внутри Telegram.

## Проверка клиента

- [ ] Клиент видит главное меню.
- [ ] Клиент может открыть запись на тренировку.
- [ ] Клиент может отправить заявку на слот.
- [ ] Клиент может открыть `Мои записи`.

## Проверка тренера

- [ ] Тренер нажал `/start`.
- [ ] Тренер видит тренерский экран.
- [ ] Тренер видит заявку клиента.
- [ ] Тренер может открыть или закрыть слоты.
- [ ] Тренер может подтвердить заявку.

## Проверка уведомлений и календаря

- [ ] Клиент получает Telegram-уведомление после подтверждения.
- [ ] Клиент получает или открывает `.ics` файл.
- [ ] В Google Calendar появляется событие.
- [ ] Отмена или перенос тренировки корректно обновляет запись.

## Что передать разработчику при проблеме

- время ошибки по Москве;
- Telegram username и Telegram ID пользователя;
- какой экран открыт;
- что нажимали;
- скриншот ошибки;
- открывалось внутри Telegram или в обычном браузере.

Не пересылать токены, пароли, приватные ключи и JSON service account в чат.
