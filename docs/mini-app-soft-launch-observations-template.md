# Soft Launch Observations Template

Использовать этот шаблон для фиксации наблюдений в первый круг soft launch.

## Observation

```md
- Soft launch observation (YYYY-MM-DD HH:MM MSK)
  - роль: клиент / тренер
  - устройство: Android / iPhone / desktop Telegram
  - сценарий: открытие / запись / подтверждение / перенос / отмена / .ics / архив / уведомление
  - результат: ok / bug
  - детали: ...
  - скрин: да / нет
  - временный обход: нужен / не нужен
```

## Daily Summary

```md
- Soft launch day summary (YYYY-MM-DD)
  - участников: ...
  - успешных сценариев: ...
  - багов-блокеров: ...
  - некритичных замечаний: ...
  - решение на следующий день: продолжать / приостановить / откатиться в основной бот
```
